## 📄 프로젝트 소개

👉 Sunny Sounds는 기상청 오픈 API를 활용한 날씨에 따른 음악을 추천해주는 사이트입니다.

- 기상청에서 불러온 데이터를 기반으로 현재 날씨를 알려줍니다.
- 날씨와 기온에 따라 추천 해주는 음악이 달라집니다.
- 또한 날씨에 따라 메인 홈 화면의 배경이 날씨에 맞게 변경이 됩니다.
